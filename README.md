\# DragonGen



DragonGen is a simulation-driven dragon colony game inspired by narrative systems like \*Wings of Fire\*.

Instead of scripted stories, DragonGen creates an evolving world where dragons form relationships, develop rivalries, experience loss, and shape the direction of their tribe over time.



\## Features



\* Dynamic dragon generation (tribe, personality, relationships)

\* Relationship system (friends, rivals, mates, family)

\* Memory system (dragons remember key events like betrayal or loyalty)

\* Leadership system (leaders influence tribe behavior and mood)

\* Tribe tension system (affects events and decision outcomes)

\* Event system with player decisions and long-term consequences

\* Encyclopedia UI with detailed dragon profiles and history



\## How It Works



Each turn, the simulation advances:



\* Relationships evolve

\* Events occur

\* Dragons may be injured, form bonds, or die

\* The tribe’s overall tension shifts based on internal dynamics



The player occasionally makes decisions that influence the world, but the system continues evolving independently.
